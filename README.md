# etlJS
The ETL capturing framework for web

# Project setup
1. Install yarn
2. Run npm install

# Dev build:
1. npm run dev

# Run test cases: 
1. npm run test

# Run production build:
1. npm run build

# Consume it:
1. var a = new etljs.SynchronousPost();
2. a.postData(url, {data});
